CAR TP 2 REST

====UTILISATION====
- La base de notre API se trouve ici : http://localhost:8080/rest/tp2/ftp/
- Pour se connecter à notre serveur il suffit d'aller ici : http://localhost:8080/rest/tp2/ftp/connect/{ip}/{port}
  Où ip est l'adresse IP du ftp et port son port choisi  lors de son démarrage.

-Pour se logger : http://localhost:8080/rest/tp2/ftp/connect/{user}/{pass}
